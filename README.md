# Quadratic Weighting as an Aggregation-Stable Power Law

Author: Georgios Kouvidis

This repository contains the simulation notebook supporting the results of the paper:

**Quadratic Weighting as an Aggregation-Stable Power Law**

## Overview

The notebook demonstrates that under additive aggregation of microscopic contributions, the effective exponent converges toward:

p ≈ 2

indicating that quadratic weighting is a structurally stable fixed point.

## Contents

- aggregation_analysis_github_ready.ipynb  
  Self-contained notebook with embedded dataset and analysis.

## Results

The notebook reproduces:

- Renormalization map R(p)
- Flow function R(p) − p
- Fixed-point identification near p ≈ 2

## Usage

Open the notebook and run all cells.

## Reproducibility

The notebook is fully self-contained and includes the dataset internally.
